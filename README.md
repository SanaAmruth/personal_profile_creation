# 🚀 Portfolio Hub: Build Your Personal Website in Seconds

**Live Demo**: [https://personal-profile-creation.onrender.com/](https://personal-profile-creation.onrender.com/)

Portfolio Hub is a powerful platform designed to help students, developers, and professionals create a stunning, high-performance personal portfolio and host it for free on GitHub Pages—all within a single form.

## ✨ Features

- **Instant Generation**: Enter your details in a simple form and get a fully-coded portfolio instantly.
- **GitHub Integration**: Connect your GitHub account and let the platform create the repository and deploy the site for you.
- **Automated Hosting**: One-click deployment to **GitHub Pages**.
- **Persistent Storage**: Securely save your profile data with Turso Cloud (SQLite).

## 🛠️ How It Works

1. **Sign Up**: Create an account to securely store your profile data.
2. **Connect GitHub**: Authenticate with GitHub so the platform can host the site for you.
3. **Fill Your Details**: Input your bio, education, experience, projects, and skills.
4. **Choose & Host**: Click "Host on GitHub" and your personal website will be live in less than 60 seconds!

---

## 💻 Tech Stack

- **Backend**: Node.js & Express
- **Database**: Turso (Cloud SQLite)
- **Frontend**: Vanilla HTML/CSS & Modern JavaScript
- **API**: GitHub REST API (Octokit)
- **Deployment**: Render
